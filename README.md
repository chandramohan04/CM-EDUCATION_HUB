# CM-EDUCATION_HUB

A full-stack web application designed for education and admissions management.

## Features
* **User Management:** Secure authentication and authorization.
* **Backend API:** Built with Node.js and Express to manage data.
* **Database:** Integrated with MongoDB.
* **Frontend:** Modern interface built with React/Vite.

## Setup Instructions

1. **Clone the repo:**
   ```bash
   git clone [https://github.com/chandramohan04/CM-EDUCATION_HUB.git](https://github.com/chandramohan04/CM-EDUCATION_HUB.git)
   cd CM-EDUCATION_HUB
   ```


2. Install dependencies:

```bash
npm install
cd frontend && npm install
```

3.Run the application:
```bash
# From the root directory

npm run dev
   ```
